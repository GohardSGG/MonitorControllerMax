# 6. 开发时间线

## 6.1 历史里程碑

- **2023.Q4**: C++ VST3 原型验证（已弃用）。
- **2024.Q1**: Rust `nih-plug` 框架迁移启动。
- **2024.Q3**: 完成基础音量控制和 GUI 框架（`egui` + `wgpu`）。
- **2024.Q4**: 
    - 实现 ZeroMQ 分布式通信。
    - 集成 Loupedeck (OSC)。
    - 发布 v2.0 Beta。
- **2025.12**: 完成 v2.5.8，实现热重载和实例级日志。

## 6.2 模块化重构 (2026.01)

**核心目标**: 解决单体架构带来的编译缓慢和依赖耦合问题。

- ✅ **Workspace 拆分**: 建立 Core, Reactor, Infra, Protocol 独立 crate。
- ✅ **Reactor 隔离**: 将 Web/OSC 服务移至独立线程，解除与 Plugin 主体的循环引用。
- ✅ **构建系统升级**: 引入 `xtask` 替代手动脚本，统一跨平台构建流程。
- ✅ **Web 控制器**: 集成 Axum 服务，实现浏览器控制原型。

## 6.3 GUI 迁移 (2026.Q1)

- ✅ **Slint 迁移**: 从 egui 迁移至 Slint 声明式 UI 框架，通过 `nih-plug-slint` 适配层集成。
- ✅ **完整 GUI 实现**: 主界面 (`home_page.slint` 73KB) 包含音量旋钮、通道面板、Solo/Mute 控制等。
- 🔲 **egui 残留清理**: `components.rs`、`scale.rs`、`keyboard_polling.rs` 等旧 egui 组件文件尚未移除（不参与编译）。
