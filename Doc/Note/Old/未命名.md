## 最终确认的完整逻辑

### 状态机

```
Idle
  │
  ├─点击 SOLO─→ SoloActive (SOLO常亮绿)
  │               │
  │               ├─点击通道─→ toggle 该通道的 Solo 状态
  │               │
  │               ├─再点 SOLO─→ Idle (清空退出)
  │               │
  │               └─点击 MUTE─→ SoloActive + MuteCompare (SOLO常亮 + MUTE闪烁)
  │                               │
  │                               ├─点击通道─→ toggle 该通道的 Mute 状态 (操作闪烁的那个)
  │                               │
  │                               ├─点击 MUTE─→ 回到 SoloActive
  │                               │
  │                               └─点击 SOLO─→ Idle (全部退出)
  │
  └─点击 MUTE─→ MuteActive (MUTE常亮红)
                  │
                  ├─点击通道─→ toggle 该通道的 Mute 状态
                  │
                  ├─再点 MUTE─→ Idle (清空退出)
                  │
                  └─点击 SOLO─→ MuteActive + SoloCompare (MUTE常亮 + SOLO闪烁)
                                  │
                                  ├─点击通道─→ toggle 该通道的 Solo 状态 (操作闪烁的那个)
                                  │
                                  ├─点击 SOLO─→ 回到 MuteActive
                                  │
                                  └─点击 MUTE─→ Idle (全部退出)
```

### 通道显示

|该通道状态|颜色|标记|
|---|---|---|
|无任何 Solo/Mute 存在|灰色|无|
|被 Solo (有声)|绿色|S|
|被静音 (无声)|红色|M|

**静音的原因：**

- 被显式 Mute
- 或者有其他通道被 Solo，所以我被 auto-mute

**一个通道永远只有一个状态，一个颜色，一个标记（或无标记）。**
