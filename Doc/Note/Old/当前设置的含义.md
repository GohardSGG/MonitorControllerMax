  当前设置的含义

  OSC Settings（OSC 设置）

  - Send Port (7444): 插件发送 OSC 消息到这个端口（给硬件控制器）      
  - Receive Port (7445): 插件监听这个端口，接收来自硬件控制器的消息    

  Network Settings（网络设置）

  - Master IP: Slave 连接 Master 时使用的地址
  - Port (9123): Master/Slave 之间 ZMQ 通信的端口

  容易产生的误解

  误解 1: "Master IP 是 Master 实例需要填写的"
  - 实际: Master IP 只对 Slave 有意义，告诉 Slave 去连接哪个 Master    
  - Master 实例会忽略这个设置

  误解 2: "OSC 端口和 Network 端口是同一个东西"
  - 实际: 完全不同的通信系统
    - OSC = 与硬件控制器（如 TouchOSC、Behringer X-Touch）通信
    - Network = 插件实例之间（Master ↔ Slave）同步状态
